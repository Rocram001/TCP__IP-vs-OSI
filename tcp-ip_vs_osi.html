OSI Model

Application
Provides network services to end users.
Presentation
Formats, encrypts, and compresses data for transmission.
Session
Manages sessions and dialogs between devices.
Transport
Ensures reliable delivery of data between devices.
Network
Handles routing and addressing of packets.
Data Link
Manages data transfer between neighboring network nodes.
Physical
Deals with hardware and transmission of raw bits.
